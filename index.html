2025

Projeto Login

Projeto Construido para treina responsividade e formulários em uma das telas mais populares em sistemas.

Para construir essa página, utilizei HTML5 para estruturar o conteúdo semântico, garantindo acessibilidade e clareza no código. O CSS3 foi aplicado para estilização avançada, incluindo variáveis de cores, bordas arredondadas, sombras, flexbox e grid para organizar os elementos de forma responsiva. Para interação, o JavaScript controla o botão “Saiba mais”, que alterna a exibição de texto extra, a troca dinâmica de tema claro/escuro e o scroll suave para navegação interna. Também usei Font Awesome para ícones, que dão suporte visual consistente e melhoram a experiência do usuário

2025

Projeto Links

Projeto que simula um celular com links para várias redes sociais, permitindo navegar entre páginas dentro do próprio dispositivo usando apenas HTML e CSS.

O principal destaque do projeto é o uso do iframe, que permite carregar diferentes páginas sem sair do layout do celular. Os ícones das redes sociais funcionam como botões que trocam o conteúdo exibido no iframe. Todo o estilo e posicionamento do celular e dos ícones são feitos com CSS, garantindo um efeito visual interativo e responsivo.

Projeto Cordel

Um site simples que apresenta a poesia “Cordel Moderno”, combinando tipografia tradicional com um layout limpo e agradável, destacando as estrofes em seções alternadas.

O projeto utiliza HTML5 para estruturar o conteúdo em seções (section) e CSS3 para aplicar o efeito de parallax nas seções com imagens de fundo. Esse efeito cria uma sensação de profundidade, mantendo o conteúdo do texto fixo enquanto o fundo se move ao rolar a página.

Projeto Android

Projeto de artigo web responsivo, que apresenta a história do mascote do Android, combinando textos, imagens e vídeos de forma organizada e visualmente atraente.

O projeto foi desenvolvido utilizando HTML semântico para estruturar conteúdos como header, main, article, aside e footer, garantindo acessibilidade e SEO. O CSS aplicou gradientes, variáveis CSS para cores e fontes personalizadas, e responsividade com max-width e media queries para imagens e vídeos. Além disso, o layout centralizado e sombras sutis conferem destaque ao conteúdo. O projeto foi realizado para pratica tags e a criação do primeiro conteúdo
